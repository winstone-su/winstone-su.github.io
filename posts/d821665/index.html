2023-07-19-Yolov5模型训练自己的数据集

发表于2023-07-19|更新于2025-03-07|AI
|总字数:1.2k|阅读时长:5分钟|浏览量:

下载yolov5

Clone repo and install requirements.txt in a Python>=3.7.0 environment, including PyTorch>=1.7.

下载模型

在https://github.com/ultralytics/yolov5往下翻，找到Pretrained Checkpoints

下载自己需要的模型,放到weight目录下(没有就新建一个)

测试自带模型

生成的结果如下，找到runs\detect\exp16 目录

可以看到这里的两张图片都已经标注完成了，但是这里多了一个labels目录，这个先不用管它，后面会讲。

标记数据集

用labelimg 标记数据集，注意格式要保存为YOLO支持的txt格式

Change save format改为YOLO

**如果已经将标注设置为PascalVOC**，也就是xml格式，可以使用以下代码转换

训练数据集

我是在yolov5的主目录下新建了一个sample/tunnel目录，专门用来存放数据集，可以自定义目录

注意这个data.yaml ，names 就是你在labelImg 标注时的标签，nc 是names的数量, train是用来训练的数据集,val 是用来验证的数据集,如果验证数据的误差过大，会重新计算。

train和val目录下都有images和labels两个目录，其中images目录是用来存放图片的，labels是用来存放标注文件的。

运行之后就是漫长的等待过程。

训练好之后，会在runs/train目录下生成训练好的文件

权重文件目录,这里的best.pt 就是训练批次最好的模型，last.pt 是最后一次的结果

测试训练好模型

结果会在runs\detect目录下

同时这里生成了一个runs\detect\exp17\labels，这个问题在上面提出过

这是因为我的需求是要求测试的模型，同时生成txt格式的标注

在参数--save-txt 添加一个默认值

这样运行detect.py 就会同时生成标注文件。

其他设置

同时如果觉得测试模型的命令太长了

可以手动修改detect.py ，可以修改--source 和--weights 为自定义的值

这样就只需要运行
